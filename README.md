## 📌 Overview

The **Advanced DTU GPA & CGPA Calculator** is a web-based academic tool developed for students of **Delhi Technological University (DTU)**.  

It enables accurate calculation of **Semester Grade Point Average (SGPA)** and **Cumulative Grade Point Average (CGPA)** based strictly on the official DTU grading system.

The application is designed to simplify complex academic calculations, especially in scenarios involving **multiple backlogs and repeated attempts**, ensuring both accuracy and ease of use.

## 💡 Problem Statement

Manual CGPA calculation becomes challenging when:

- Subjects are repeated due to backlogs  
- Multiple attempts exist for the same subject  
- Previous grades must be replaced correctly  
- Multi-semester aggregation is required  

According to DTU rules, **only the latest valid attempt is considered**, which makes manual computation error-prone.

---

## ✅ Solution

This tool provides a fully automated system that:

- Identifies and replaces outdated attempts  
- Considers only valid grades as per DTU policy  
- Handles multi-semester data seamlessly  
- Eliminates manual calculation errors  

The result is a **fast, reliable, and accurate CGPA calculation experience**.

## 🚀 Features

- Accurate SGPA & CGPA calculation (DTU official system)  
- Multi-semester CGPA computation

## 📌 How It Works

1. Enter subject details along with assigned credits  
2. Select grades based on the DTU grading system  
3. SGPA is calculated instantly (real-time)  
4. Add multiple semesters to compute CGPA  
5. Download the final result as a PDF  

---

## 📊 Grading System

This tool follows the official **DTU Absolute Grading System (10-point scale)** to ensure accurate and consistent results.
